# Обзор проекта

Краткое описание:
Проект — PHP-приложение Telegram-бота с интеграцией платёжного шлюза FreedomPay и обработкой вебхуков.

Ключевые файлы:
- `index.php` — точка входа / общий роутер.
- `TelegramBot.php` — логика взаимодействия с Telegram.
- `FreedomPay.php` — интеграция с платёжным сервисом (создание платежей, валидация уведомлений).
- `payment_webhook.php` — приём уведомлений о платеже.
- `mock_freedompay.php` — тестовый имитатор платёжного провайдера.
- `database.php` — работа с базой данных.

Цели ТЗ-файлов:
- Описать архитектуру, API, схему БД, сценарии бота и требования к развертыванию.

Контекст среды разработки: проект разработан для локальной среды (OSPanel/Windows), но может быть развернут на Linux-сервере.
